## Project Overview

This collaborative project is a modern and interactive **Movie Review Website**, allowing users to browse movie details, view posters, and read reviews. The content is managed through a Headless CMS and dynamically rendered via a responsive frontend.

---

## Technology Stack

### Headless CMS Repository

- **Tool**: [Strapi](https://strapi.io/)  
- **Purpose**: Store and manage movie data, reviewer details, poster images, and metadata  
- **Repository Type**: Private GitHub repository  

### Frontend Repository

- **Tool**: [Vue.js](https://vuejs.org/) with [Vite](https://vitejs.dev/)  
- **Styling**: [Bootstrap](https://getbootstrap.com/) for responsive layout and UI components  
- **Purpose**: Display dynamic content, implement routing and search, and provide an accessible user experience  
- **Repository Type**: Private GitHub repository
